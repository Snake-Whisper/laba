Laba-Server DEV Version
===========

This server implements a simple chat server app written in python/flask.

Installation:
-------------
```bash
#please config laba/config
nano laba/config

#run installer
./install.sh
```
Start Server:
------------
```bash
./start.sh
```

Docker:
-------
```bash
docker-compose up
```

surf at http://localhost:8080
